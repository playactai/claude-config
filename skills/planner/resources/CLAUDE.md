# resources/

Planner-specific reference files. `plan-format.md` is the human-readable plan-structure reference; the architect reads `plan-json-schema.md` at runtime for the authoritative JSON schema.

## Files

| File                       | What                                          | When to read                   |
| -------------------------- | --------------------------------------------- | ------------------------------ |
| `plan-format.md`           | Plan structure reference (human-readable)     | Editing plan structure         |
| `plan-json-schema.md`      | Authoritative JSON schema (read by architect) | Modifying plan.json schema     |
| `explore-output-format.md` | XML schema for explore output                 | Modifying exploration workflow |
| `README.md`                | Loading mechanism                             | Understanding resource loading |
